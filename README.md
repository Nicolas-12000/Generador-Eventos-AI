# 🎉 Generador AI de Landing Pages para Eventos (Frontend)

Aplicación React que interactúa con un backend Spring Boot + Azure OpenAI para generar landing pages personalizadas mediante un chat intuitivo.

## 🚀 Tecnologías
- **Frontend**: React 18 + Vite
- **Estilos**: Tailwind CSS
- **Routing**: React Router v6
- **HTTP Client**: Axios
- **Mapas**: Google Maps JavaScript API
